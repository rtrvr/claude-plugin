`$ARGUMENTS`를 기반으로 Metabase 분석을 도와준다.

- `skills/metabase/SKILL.md`를 읽어 워크플로우를 따른다
- 스키마가 필요하면 `skills/metabase/references/schema.md`를 읽는다
- 인자가 비어있으면 `AskUserQuestion`으로 "어떤 데이터를 조회하거나 시각화하고 싶으신가요?"를 질문한다
- 모든 응답은 한국어로 작성한다
